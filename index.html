Courant News

The open source CMS designed specifically for college newspapers

Project Discontinued

After over 18 months of work, development on the Courant News project was discontinued in February 2010.

The project source code is available on GitHub for those interested. The original authors of Courant News have moved on to new projects; see their personal websites or blogs for more information.
